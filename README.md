## hello-rails-react

> Display the random greeting on a button click

## Setup
>- Git clone `https://github.com/rolandfbk/hello-rails-react.git`
>- cd hello-rails-react
>- bundle install
>- rails db:create db:migrate
>- yarn install
>- yarn build
>- rails server

## Built With
>- Webpack / React
>- Rails 7.0
